# 过渡动画

### 进入/离开&列表过渡

简介：vue在插入、更新、移除dom时候，提供不同方式的引用过渡效果

* css过渡和动画自动应用
* 配合第三方css库，例如：Animaate.css
* 在过渡钩子函数中使用JavaScript直接操作dom
* 可以配合第三方JavaScript动画库，例如：velocity.sj

###   单元素/组件的过渡

`transiton`组件，在以下情境中，可以给任何元素添加进入、离开动画

1. 条件渲染v-if
2. 条件展示v-show
3. 动态组件
4. 组件根节点
